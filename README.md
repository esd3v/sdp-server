> Scrapes and parses discussion topics of requested steam app. Server-side.
## Prerequisites

- node 12.8.1
- npm 6.10.2

## Install

```sh
npm install
```

## Watch

```sh
npm run watch
```

## Launch

```sh
npm run start
```

## Run tests

```sh
npm run test
```
